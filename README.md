# HackDtu2.0

Idea

Locating and providing details of child care centers and nutrition rehabilitation centers in your area with the help of a webapp/mobile app. It will analyze current type of vitamin deficiency faced by infants. Data collected will be used to predict upcoming nutritional requirements of that particular district, so that required vitamin supplements and medicines could be restocked in advance. This will help will in combating vitamin deficiency in an efficient way thereby increasing infant mortality rate and avoiding catastrophic events. 

Implementation

Spreading the application to different NGO’s, indivisuals and government organizations working for this cause. It will be available as an app or web page.
